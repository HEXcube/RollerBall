# RollerBall

Simple game demo I created while trying out [Roll-a-Ball](https://learn.unity.com/project/roll-a-ball) tutorial on Unity Learn.


HOWTO
-----
- Requires Unity 2019.4 LTS
- Clone this project, import in Unity and build project
